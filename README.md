# Subtitle Viewer & Controller

## General Information
This is a small project that allows users to **open already downloaded subtitles for movies** and provides **basic control and handling of subtitle files**.

## Features
- Open and view subtitle files (.srt)  
- Control subtitle playback: **play, pause,moev backward/forward**  
- GUI auto hide after a set period of inavctivity
- Timer
- Simple and intuitive GUI for quick subtitle management

## Software
- Python 3.x

## How to use

1. Launch the program.  
2. When the timer appears at the bottom of the screen, hover over it to reveal the GUI in the top-right corner.  
3. Load a movie:
   - Click the **directory icon** button.  
   - Browse and select the movie file you want to open.  
4. Click **Play** to start subtitle playback.  
5. Use the playback controls to:
   - **Play** or **Pause** the subtitles  
   - **Move backward/forward by 10 seconds**  
6. The GUI will automatically hide after a set period of inactivity (timer functionality).
